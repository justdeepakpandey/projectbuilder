# 🚀 AI Project Builder

A full-stack MERN application that allows users to generate and manage projects dynamically with authentication support.

## ✨ Features

- User Signup & Login Authentication
- Generate AI-based project ideas
- Save projects to MongoDB
- Edit projects
- Delete projects
- Persistent storage after refresh
- Responsive UI
- Full-stack deployment

## 🛠 Tech Stack

### Frontend
- React.js
- Vite
- CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas

### Deployment
- Frontend: Vercel
- Backend: Railway

## ⚙ Installation

Clone repository:

```bash
git clone https://github.com/justdeepakpandey/projectbuilder.git
```

Go into project:

```bash
cd projectbuilder
```

Install frontend dependencies:

```bash
cd client
npm install
npm run dev
```

Install backend dependencies:

```bash
cd server
npm install
node server.js
```
